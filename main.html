Qustion-3:What is event delegation in the DOM?

Event Delegation is a technique where an event listener is added to a parent element to handle events on its child elements using event bubbling.

Qustion-4: What is event bubbling in the DOM?

Event Bubbling is a DOM event propagation method where an event starts from the target element and bubbles up to its ancestors (parent, grandparent, etc.).
